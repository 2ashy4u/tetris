# Requirements

- Qt >= 6.0.0
- OpenGL
- CMake >= 3.5
- C++14

# Building/Executing

Steps from cloning the repository to running the game:

```bash
$ cd tetris
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./Tetris
```

From this step, you can generate the documentation:

```bash
$ cd doc
$ make
$ firefox ../../doc/html/index.html
```
